# 📝 Blog Platform

A modern Full Stack Blog Platform built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). Users can register, log in, create blogs, manage their posts, and explore content through a clean and responsive interface.

---

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- ✍️ Create, Edit and Delete Blog Posts
- 📖 Read Blog Posts
- 👤 User Profile Page
- 📊 Dashboard for Managing Posts
- 🔍 Search Blogs
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive Design
- ⚡ Fast React Frontend
- 🔒 JWT Authentication (if implemented)

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/gkavya976/Blog_platform.git
```

Move into the project folder

```bash
cd Blog_platform
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

---

## Frontend Setup

Open another terminal.

```bash
cd frontend
npm install
npm start
```

Frontend will run at

```
http://localhost:3000
```

Backend will run at

```
http://localhost:5000
